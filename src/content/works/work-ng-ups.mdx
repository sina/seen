<p class="mb-8">Nasty Gal is an online apparel website specializing in affordable
  fast-fashion items for young women aged 16 - 30.</p>

<p>Integrated with UPS® shipping service APIs to allow Nasty Gal customers to
  perform self-service returns and instantly print UPS® return shipping
  labels. The node/express webapp created and processed xml requests adhering
  to the UPS® schema allowing customers to quickly exchange items and promote
  frequent purchases.</p>
