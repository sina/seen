<p class="mb-8">This recruitment platform was designed to streamline the hiring
  process by providing job listings, applicant collection/dispositioning,
  screening and communication.</p>
  
<p>Extended functionality of a Spring MVC and Java backed recruitment web
  application. Created a telephone questionnaire using Asterisk PBX and
  Java services to record questionnaire, call/email applicants, record responses,
  and display audio associated with applications in job listing dashboard.</p>
